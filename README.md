# rising3.github.io

RISING3 GitHub Pages

This is a [RISING3 GitHub Pages](https://rising3.github.io/).

## Getting Started

First, run the development server:

```bash
npm i
npm run dev
```
Open [http://localhost:4321](http://localhost:4321) with your browser to see the result.
